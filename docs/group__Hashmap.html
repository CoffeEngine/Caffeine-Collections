Initializes the hashmap structure. 

This functions allows the user to improve the hashmap for his use case, possibiliting to set callbacks that better handles the key.
Parameters
hashmap[in] Pointer to a hashmap.
key_size[in] Key size in bytes.
data_size[in] Value size in bytes
lenght[in] How many elements will be allocated initialy
hash_func[in] Pointer to a hash function callback
cmp_func[in] Pointer to a comparer function callback
cpy_func[in] Pointer to a copy function callback
allocator[in] Pointer to a custom allocator. If NULL is setted the function will use the default allocator from Caffeine-Core library.
Returns
cff_err_e A cff_err_e indicating if any error ocurred.
