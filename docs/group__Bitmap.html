cff_err_e cff_bitmap_resize ( cff_bitmap *  bmp,
uint64_t  lenght,
AllocatorInterface *  allocator 
)

Resizes the bitmap buffer.

This function reallocates the buffer who can have it's memory position changed. If the new lenght requires a buffer with same size than current buffer the function just returns.

Parameters
bmp[in] Pointer to a bitmap.
lenght[in] New lenght.
allocator[in] Pointer to a custom allocator. If NULL is setted the function will use the default allocator from Caffeine-Core library.
Returns
cff_err_e A cff_err_e indicating if any error ocurred.
